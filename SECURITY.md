# Security Policy

## Supported versions
Users can expect that past 2024 June 26, [`trunk`](https://github.com/SwuduSusuwu/SubStack) passes [`susuwuUnitTests`/`Susuwu::unitTestsCxx`](https://github.com/SwuduSusuwu/SubStack/blob/trunk/cxx/main.cxx#L22-L82) + [_CodeQL_](https://github.com/SwuduSusuwu/SubStack/security/code-scanning)

## Sensitive issues
First view [How to contribute](https://github.com/SwuduSusuwu/SubStack/blob/trunk/README.md#How-to-contribute) for information on issues (to ensure that what you found is not a normal issue).
If you found [normal issue(s)](https://github.com/SwuduSusuwu/SubStack/security/code-scanning/1882), use [this normal route to post about new issues](https://github.com/SwuduSusuwu/SubStack/issues/new),
but if you found [sensitive issue(s)](https://github.com/SwuduSusuwu/SubStack/security/code-scanning/1277), report the issue through private message to "Swudu Susuwu", or [mailto:2002luvabbaluvu@gmail.com](mailto:2002luvabbaluvu@gmail.com).

You can expect:
    Best effort to address the issue(s),
    with you anonymous (unless you ask to publish credits to you.)
